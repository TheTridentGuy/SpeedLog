About:

SpeedLog is a amateur radio logging program. It is available for use for free under GNU GPL-V3. The source code is open-source on the SpeedLog Github.

Important Notice:

Logs made with speedlog may not be encrypted, and can be accessed, destroyed, corrupted, etc at any time by whoever is hosting this instance of SpeedLog. You alone are responsible for backing up your logs, which can be exported from the viewer. It is also strongly recommended that you avoid putting any potentialy sensitive information in your logs.

More detailed instructions available on the SpeedLog Github
